Privacy Policy

1. Collected Personal Information & Purpose

The Service collects only the minimum information necessary to provide a smooth service (such as data saving and cross-device synchronization).

  • Collected Items: Email address (optional), password (stored as a one-way hash), nickname, service usage history (saved calculation data, etc.)
  • Purpose: User identification, password recovery, personalized data saving and synchronization

2. Retention & Usage Period

Collected personal information is retained and used until the user requests account deletion. Upon account deletion via the 'Delete Account' feature in the Service, all data is immediately and irreversibly destroyed.

3. Third-Party Sharing & Outsourcing

The Service does not provide user personal information to any third parties nor outsource data processing to external entities.

4. Data Destruction Process

Users may delete their account at any time from the account profile page. Upon a deletion request, the user identification information stored on the server and all synchronized calculation data are immediately removed.
